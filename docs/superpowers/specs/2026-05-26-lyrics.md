# Spec: Lyrics Feature

Fetch lyrics from LRCLIB, embed directly into audio file metadata tags.

## Decisions

| # | Decision | Choice |
|---|----------|--------|
| 1 | When to fetch | Auto post-download (default). Configurable. On-demand bulk fetch available. |
| 2 | Library | `lrcup` â€” `LRCLib` (API client) + `AudioFile` (mutagen wrapper) |
| 3 | Format | Synced LRC preferred; plain text fallback. Stored in file tag only. |
| 4 | DB tracking | `lyrics_status` column â€” 5 values (see below) |
| 5 | Lookup strategy | `get()` first â†’ fallback `search()` â†’ first result |
| 6 | Network errors | Keep `not_fetched`; 404/empty â†’ `not_found` |
| 7 | Service location | `src/services/lrclib_service.py` â€” class `LRCLibService(conn)` |
| 8 | API methods | `fetch_lyrics`, `fetch_missing_lyrics`, `get_lyrics` |
| 9 | Config | `auto_fetch_lyrics = true` in DEFAULTS |
| 10 | Orchestration | `api.py` passes `on_complete` hook to `queue_download()` |
| 11 | Bulk fetch threading | Background daemon thread |
| 12 | Bulk fetch events | `lyrics_progress` per track + `lyrics_fetch_complete` with summary |
| 13 | `get_lyrics` source | Reads from audio file tag via `lrcup.AudioFile` |
| 14 | DB migration | Add column directly to `CREATE TABLE tracks` â€” no existing DB |



`_row_to_track` must include `lyrics_status` in the returned `Track` object.

## 4. `src/services/lrclib_service.py`

### Class: `LRCLibService(conn)`

```python
class LRCLibService:
    def __init__(self, conn: sqlite3.Connection) -> None
    def fetch_and_embed(self, track_id: int) -> None
    def fetch_missing_lyrics(self) -> None        # starts background thread
    def get_lyrics(self, track_id: int) -> str | None
```

### `fetch_and_embed(track_id)`

1. Load track from DB. If not found or `file_path` is None â†’ return early.
2. Try `LRCLib().get(track=title, artist=artist, album=album, duration=duration)`:
   - Skip `get()` if `duration` is None â†’ go directly to step 3.
   - On network/HTTP error â†’ log, set status `not_fetched`, return.
3. If `get()` returns None â†’ try `LRCLib().search(track=title, artist=artist)` â†’ take `results[0]` if any.
   - On network/HTTP error â†’ log, set status `not_fetched`, return.
4. If no result after both â†’ `update_lyrics_status(conn, track_id, "not_found")`, return.
5. If `result.instrumental` â†’ `update_lyrics_status(conn, track_id, "instrumental")`, return.
6. Determine lyrics and state:
   - `syncedLyrics` not None â†’ `lyrics = syncedLyrics`, `state = "synced"`, `status = "synchronized"`
   - else `plainLyrics` not None â†’ `lyrics = plainLyrics`, `state = "unsynced"`, `status = "plain_text"`
   - else â†’ treat as `not_found`
7. `AudioFile(Path(track.file_path)).set_lyrics(state=state, lyrics=lyrics)`
   - On `UnsupportedSuffix` â†’ log, set status `not_fetched`, return.
8. `update_lyrics_status(conn, track_id, status)`

### `fetch_missing_lyrics()`

Starts a background daemon thread that:
1. Loads all tracks with `lyrics_status = "not_fetched"` from DB.
2. For each track:
   - Calls `fetch_and_embed(track.id)`
   - Emits `event_bus.emit("lyrics_progress", {"track_id": track.id, "status": <new_status>})`
3. After all done, emits:
   ```python
   event_bus.emit("lyrics_fetch_complete", {
       "synchronized": int,
       "plain_text": int,
       "instrumental": int,
       "not_found": int,
       "errors": int,   # tracks that stayed not_fetched due to errors
   })
   ```

### `get_lyrics(track_id)`

1. Load track from DB. If not found or `file_path` is None â†’ return None.
2. `AudioFile(Path(track.file_path)).get_lyrics()` â€” return result (may be None).
3. On any exception â†’ log, return None.

## 5. Changes to `src/services/youtube_service.py`

`queue_download` signature change:

```python
def queue_download(self, track_id: int, on_complete: Callable[[int], None] | None = None) -> None
```

In `download()`, after `event_bus.emit("download_complete", ...)`:
```python
if on_complete:
    on_complete(track_id)
```

`on_complete` is NOT called on failure.

## 6. Changes to `src/api/api.py`

### Lazy initializer

```python
def _get_lrclib(self) -> LRCLibService:
    if self._lrclib is None:
        self._lrclib = LRCLibService(self._conn)
    return self._lrclib
```

### `queue_download` wiring

```python
def queue_download(self, track_id: int) -> str:
    auto = get_setting(self._conn, "auto_fetch_lyrics") == "true"
    hook = self._get_lrclib().fetch_and_embed if auto else None
    self._get_youtube().queue_download(track_id, on_complete=hook)
    return self._ok()
```
